# WePicText

WePicText 是一个微信公众号贴图文字整理浏览器插件。

公众号贴图内容的图片是左右滑动合集，文字单独放在图片下方。这个文字区不能调字号，也不能加粗，所以 WePicText 只解决一个问题：把 `.txt` 或 `.md` 草稿整理成适合粘贴到贴图文字区的干净正文。

## 功能

- 拖入 `.txt` 文件
- 拖入 `.md` 文件后自动清理 Markdown 符号
- 去除 `#`、`**`、图片引用、分隔线等多余格式
- 自动整理段落
- 在段落之间加入适合公众号贴图文字区的空行
- 复制整理后的正文，再手动粘贴到公众号贴图文字区

## 下载哪个文件

普通用户下载这个文件：

```text
WePicText-0.1.7.zip
```

下载后先解压，得到 `WePicText-0.1.7` 文件夹，再在浏览器里加载这个解压后的文件夹。

不要直接选择压缩包安装，Chrome / Arc 加载本地插件时需要选择解压后的文件夹。

## 安装

1. 下载 `WePicText-0.1.7.zip`。
2. 解压这个 zip。
3. 打开 Chrome / Arc 的扩展程序管理页。
4. 打开开发者模式。
5. 选择“加载已解压的扩展程序”。
6. 选择解压后的 `WePicText-0.1.7` 文件夹。
7. 打开微信公众号贴图编辑页。
8. 使用右下角 WePicText 面板拖入 `.txt` 或 `.md`。
9. 点击“复制并定位”。
10. 在公众号文字区粘贴：Mac 按 `⌘V`，Windows 按 `Ctrl+V`。

## 开发者安装

1. 克隆本仓库。
2. 打开 Chrome / Arc 的扩展程序管理页。
3. 打开开发者模式。
4. 选择“加载已解压的扩展程序”。
5. 选择本仓库文件夹。
6. 打开微信公众号贴图编辑页。
7. 使用右下角 WePicText 面板拖入 `.txt` 或 `.md`。
8. 点击“复制并定位”。

## 边界

- 不上传图片
- 不处理图片排序
- 不自动发布内容
- 不修改公众号账号设置
- 不把文章内容发送到任何服务器

## 隐私

WePicText 不收集、不上传、不出售任何用户数据。所有文字清理都在浏览器本地完成。
